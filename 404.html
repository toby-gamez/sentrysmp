What is 404?

A "404" is an HTTP status code that indicates a webpage could not be found on the server. It typically occurs when a user tries to visit a URL that doesn't exist or has been moved. The server returns a 404 error when it cannot locate the requested page, often accompanied by a message like "404 Not Found" or a custom error page indicating that the page is missing. This is a common error seen on websites.
